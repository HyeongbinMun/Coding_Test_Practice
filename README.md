## Anouncement

1. 메세지 규약: [문제 출처 사이트 약자]/ 문제 번호/ 난이도

    - ex) [BOJ] / 1439 / bronze

    - (백준:BOJ, 프로그래머스:PMS, 릿코드:LCD)

2. 파일 형식: .py(문제 당 1개)

3. 주석: 문제 링크, 걸린 시간, 사용한 메모리 등

    - ex) #문제링크: https://www.acmicpc.net/problem/1439
    - ex) #걸린시간: 0.7s
    - ex) #사용한 메모리: 50MB
